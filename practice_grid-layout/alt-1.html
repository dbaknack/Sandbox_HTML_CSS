SQL Server Sessions

Overview

The SQL Server Sessions chart provides a breakdown of active and idle sessions, login types, and session durations. It helps database administrators track performance and optimize resources effectively.

Key Metrics

  • Active Sessions: Displays sessions currently executing queries.
  • Idle Sessions: Sessions connected but not running queries.
  • Session Duration: Tracks the length of each session.
  • Login Types: Differentiates between Windows and SQL logins.

Use this chart to identify long-running queries and idle connections. Highlight sessions with long durations for optimization.
